## Notes
- **Sandbox Philosophy**: All languages must run locally using free, open-source interpreters/compilers
- **No Cloud Dependencies**: Everything runs on the user's machine - no external API calls for code execution
- **Auto-Detection**: System should detect available languages and guide users to install missing ones
- **Security First**: Each language execution is sandboxed with timeouts and resource limits
- Prioritize features based on user feedback
- Consider performance impact of each feature
- Maintain backward compatibility
- Keep UI clean and intuitive
- Focus on user value over feature count

## Language Runtime Requirements (for reference)
All languages use local, free interpreters/compilers:
- **Python**: python3 (usually pre-installed on Linux/Mac)
- **JavaScript**: node (Node.js - free, open-source)
- **PHP**: php (free, open-source)
- **Ruby**: ruby (free, open-source)
- **Go**: go (free, open-source from Google)
- **Rust**: rustc/cargo (free, open-source)
- **C/C++**: gcc/clang (free, open-source)
- **Java**: OpenJDK (free, open-source)
- **C#**: .NET SDK (free, open-source from Microsoft)
- **TypeScript**: ts-node (npm package)
- **R**: R (free, open-source)
- **Lua**: lua (free, open-source)
- **Perl**: perl (usually pre-installed)
- **Bash**: bash (usually pre-installed)
- **Swift**: swift (free, open-source from Apple)
- **Kotlin**: kotlinc (free, open-source from JetBrains)
- **Julia**: julia (free, open-source)
- **Dart**: dart (free, open-source from Google)
- **Haskell**: ghc (free, open-source)
- **Scala**: scala (free, open-source)
- **Clojure**: clojure (free, open-source)
- **Elixir**: elixir (free, open-source)
- **Erlang**: erlang (free, open-source)
